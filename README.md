## 4. 적용 기술 및 구현 기능

### 4-1. 적용 기술

- Back-end : JavaScript, Node.js, Express, Bcrypt, JWT
- Database : MySQL
- HTTP : Postman, live share
- Common : Slack, Zep, Notion, Git&Github

### 4-2. 구현 기능

- 김교은

  - 회원가입/로그인
  - 이용권
  - 플레이리스트 CRUD
  - SMS 인증 (네이버 SENS api 연동)

- 이신희
  - 메인, 둘러보기, 보관함, 디테일 페이지 API
  - 곡 재생 플레이리스트 추가, 좋아요 기능 API
